#  متجرنا الإلكتروني

واجهة متجر إلكتروني عصرية وبسيطة، مبنية باستخدام **React** و **Material UI**.

##  المميزات
* **تصميم أنيق:** واجهة مستخدم مريحة وهادئة.
* **متجاوب:** تصميم مرن يعمل على جميع أحجام الشاشات (Mobile, Tablet, Desktop).
* **بحث وفلترة:** نظام بحث وتصنيف للمنتجات سهل الاستخدام.

##  التقنيات المستخدمة
* **React.js**
* **Material UI (MUI)**
* **React Router**

##  تشغيل المشروع

1. تحميل المشروع:

```bash
git clone https://github.com/AbdullahAbuZaid04/e-commerce.git
```

2. تثبيت المكتبات:

```bash
npm install
```

3. تشغيل المشروع:

```bash
npm start
```
